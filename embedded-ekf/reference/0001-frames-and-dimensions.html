Reference 0001: Frames And Dimensions

A compact reference for the course's first contract.

Frames

Frame Axes Course meaning
Local NED x north, y east, z down Navigation frame fixed at the launch pad.
Body x_b nose-forward, y_b right, z_b down Rocket-fixed frame used by IMU measurements.

State Dimensions

Constant Value Reason
ESKF_VEC3_DIM 3 North/east/down or body x/y/z vector.
ESKF_QUAT_DIM 4 Quaternion scalar/vector storage.
ESKF_NOMINAL_STATE_DIM 16 p[3] + v[3] + q[4] + b_g[3] + b_a[3].
ESKF_ERROR_STATE_DIM 15 dp[3] + dv[3] + dtheta[3] + dbg[3] + dba[3].
ESKF_IMU_NOISE_DIM 12 Gyro noise, accel noise, gyro-bias random walk, accel-bias random walk.
ESKF_GPS_POS_VEL_DIM 6 GPS position plus velocity in local NED.
Key rule: The quaternion has four stored values, but the attitude error has three values. That distinction is the core reason this course is an ESKF course rather than a plain state-vector EKF course.
