Embedded ESKF Glossary

Short definitions used consistently across the course.

ESKF
Error-state Kalman filter. The nominal state carries the best estimate, while a small error state carries uncertainty and corrections.
Nominal state
The estimator's main state: position, velocity, attitude quaternion, gyro bias, and accelerometer bias.
Error state
The small correction state: position error, velocity error, 3D attitude error, gyro-bias error, and accelerometer-bias error.
Local NED
A local navigation frame fixed at the launch pad where x is north, y is east, and z is down.
Body frame
The rocket-fixed frame where x_b points along the nose, y_b points right, and z_b points down.
q_nb
The attitude quaternion that rotates body-frame vectors into the local navigation frame.
Specific force
The accelerometer measurement after bias correction. It is not simply vehicle acceleration because gravity handling depends on the frame and model.
Covariance
A matrix that represents uncertainty and correlation in the error state, not a direct copy of the nominal state.
Gyro bias
A slowly changing angular-rate offset, stored in body-frame rad/s, subtracted from raw gyro measurements before attitude prediction.
Accelerometer bias
A slowly changing specific-force offset, stored in body-frame m/s^2, subtracted from raw accelerometer measurements before acceleration prediction.
Skew matrix
A 3x3 matrix form of the cross product. It lets attitude-error terms appear inside ordinary matrix equations.
F matrix
The 15x15 linearized error-dynamics matrix used to predict how small errors move through time.
G matrix
The 15x12 noise-input matrix that maps IMU measurement noise and bias random walk into the error state.
Random walk
A model where uncertainty grows over time because the underlying value can drift unpredictably, commonly used for IMU bias evolution.
