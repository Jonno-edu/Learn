Quaternion Basics

norm(q) should stay near 1

Use scalar-first layout. Multiplication composes rotations and is not commutative.

Vector Rotation

v_n = rotate(q_nb, v_b)

q_nb rotates body-frame vectors into the local NED navigation frame.

IMU Prediction Formula Set

omega_b = gyro_rad_s - b_g
f_b     = accel_mps2 - b_a

q_nb = integrate(q_nb, omega_b, dt_s)

g_n = [0, 0, 9.80665]
a_n = rotate(q_nb, f_b) + g_n

p_n = p_n + v_old * dt_s + 0.5 * a_n * dt_s * dt_s
v_n = v_n + a_n * dt_s

Skew Matrix

skew([ax, ay, az]) =
[  0  -az   ay
   az    0  -ax
  -ay   ax    0 ]

This matrix appears in linearized attitude and acceleration coupling.
