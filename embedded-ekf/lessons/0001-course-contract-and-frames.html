Lesson 0001: Course Contract And Frames

This lesson locks the first estimator contract: which frames exist, which quantities are measured, and which dimensions the C code must never blur.

Back to course index

The One Thing

Before writing estimator math, we need a shared contract. Every vector, quaternion, covariance block, and measurement must have an agreed frame, unit, and dimension.

This is not paperwork. In an ESKF, a wrong frame can look like a tuning problem, a wrong sign can look like a sensor problem, and a wrong matrix dimension can still compile if the code is too generic. Lesson 1 prevents those failures before the filter grows.

The habit to build: when you say p, v, q_nb, P, or gps, you should be able to say its frame and dimension immediately.

Frames

Frame Axes Use
Local NED x north, y east, z down Navigation frame fixed at the launch pad.
Body x_b nose-forward, y_b right, z_b down Rocket-fixed frame used by the IMU.

The attitude quaternion is named q_nb. In this course it rotates body-frame vectors into the navigation frame. 

v_n = rotate(q_nb, v_b)

Reference: VectorNav has a concise inertial-navigation primer on reference frames.

State Contract

State Pieces Dimension
Nominal p_n[3], v_n[3], q_nb[4], b_g[3], b_a[3] 16 stored values
Error dp[3], dv[3], dtheta[3], dbg[3], dba[3] 15 correction values
The attitude quaternion stores four values, but the attitude error is a three-value small-angle vector. That is the first major ESKF distinction.

Reference: Joan Sola's quaternion ESKF paper is the course's main source for attitude-error handling.

Why The Dimensions Are The First Code

The dimension constants become the shared vocabulary for every later function. They will decide the size of P, F, G, GPS residuals, Kalman gain matrices, and correction vectors.

Later object Dimension source
P ESKF_ERROR_STATE_DIM x ESKF_ERROR_STATE_DIM
H_gps_pos_vel ESKF_GPS_POS_VEL_DIM x ESKF_ERROR_STATE_DIM
K_gps_pos_vel ESKF_ERROR_STATE_DIM x ESKF_GPS_POS_VEL_DIM
G ESKF_ERROR_STATE_DIM x ESKF_IMU_NOISE_DIM
