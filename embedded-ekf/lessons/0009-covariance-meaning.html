Lesson 0009: Covariance Meaning

Covariance is the filter's uncertainty model. It tells the update step how much to trust prediction versus measurement.

The One Thing

In this ESKF, covariance P is 15x15 because it belongs to the error state: 

x_error = [dp, dv, dtheta, dbg, dba]

Diagonal entries are variances. Off-diagonal entries are correlations between errors.

Block Layout

Index rangeBlockMeaning
0..2dpPosition error variance and correlation.
3..5dvVelocity error variance and correlation.
6..8dthetaSmall attitude error variance and correlation.
9..11dbgGyro-bias error variance and correlation.
12..14dbaAccelerometer-bias error variance and correlation.
A covariance matrix should remain symmetric. If prediction or update makes P[i][j] and P[j][i] drift apart, that is a numerical health signal.

C Target

EskfStatus mat15_zero(eskf_real_t P[ESKF_ERROR_STATE_DIM][ESKF_ERROR_STATE_DIM]);
EskfStatus mat15_set_diagonal(eskf_real_t P[ESKF_ERROR_STATE_DIM][ESKF_ERROR_STATE_DIM],
                              const eskf_real_t diag[ESKF_ERROR_STATE_DIM]);
bool mat15_is_symmetric(const eskf_real_t P[ESKF_ERROR_STATE_DIM][ESKF_ERROR_STATE_DIM],
                        eskf_real_t tolerance);

Tests should check diagonal initialization, symmetry detection, and rejection of negative variance config values.

Quick Check

What does a large position variance mean?

It means the filter believes the position error could be large. A GPS position measurement will usually get more weight when prediction uncertainty is high relative to measurement noise.
