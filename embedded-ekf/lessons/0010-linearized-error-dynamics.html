Lesson 0010: Linearized Error Dynamics

Prediction moves the nominal state. The covariance prediction needs a linearized model of how small errors move around that nominal state.

The One Thing

The matrix F describes how the 15D error state changes over time. The matrix G describes how IMU noise enters that error state. 

F: 15x15
G: 15x12
noise: [gyro_noise, accel_noise, gyro_bias_rw, accel_bias_rw]

This lesson is not about memorizing a wall of equations. It is about making each nonzero block explainable.

Continuous-Time Error Relationships

Error derivativeMain dependenciesIntuition
d(dp)/dtdvPosition error grows from velocity error.
d(dv)/dtdtheta, dba, accel noiseWrong attitude or accel bias gives wrong acceleration.
d(dtheta)/dtdtheta, dbg, gyro noiseWrong gyro bias drives attitude drift.
d(dbg)/dtgyro-bias random walkBias uncertainty can grow over time.
d(dba)/dtaccel-bias random walkBias uncertainty can grow over time.

Course V1 Block Contract

With corrected body angular rate omega_b, corrected specific force f_b, and rotation matrix R_nb from q_nb, the important continuous-time blocks are: 

F[p, v]         = I
F[v, dtheta]    = -R_nb * skew(f_b)
F[v, dba]       = -R_nb
F[dtheta,dtheta]= -skew(omega_b)
F[dtheta,dbg]   = -I

The chosen noise order for G is: 

0..2   gyro measurement noise
3..5   accel measurement noise
6..8   gyro-bias random walk
9..11  accel-bias random walk
Sign conventions must be tested. This page defines the course v1 contract, and later numerical tests should protect it.

Quick Check

Why does F[p, v] equal identity?

Position is the integral of velocity. A small velocity error directly creates a position error over time in the same navigation-axis direction.
