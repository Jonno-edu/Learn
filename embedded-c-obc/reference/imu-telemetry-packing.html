IMU Telemetry Packing

A reference for turning typed IMU data into a bounded OBC telemetry frame.

Contract Pattern

bool pack_imu_packet(const ImuSample *imu,
                     FlightState state,
                     uint8_t *out_frame,
                     size_t out_capacity,
                     size_t *out_len);
const ImuSample *imuRead-only typed sensor data.
FlightState stateEnum value encoded into one payload byte.
uint8_t *out_frameWritable byte storage owned by the caller.
size_t out_capacityNumber of bytes valid in out_frame.
size_t *out_lenWritable result slot for the number of frame bytes produced.

Frame Layout

Offset Field
00xa5 SOF0
10x5a SOF1
20x42 message ID
313 payload length
4flight state
5..16six int16_t IMU fields, little-endian
17checksum

Little-Endian Helper

uint16_t bits = (uint16_t)value;
dst[0] = (uint8_t)(bits & 0xffu);
dst[1] = (uint8_t)((bits >> 8u) & 0xffu);

Check pointer arguments and output capacity before writing the frame. Set *out_len only after the complete frame has been written.
