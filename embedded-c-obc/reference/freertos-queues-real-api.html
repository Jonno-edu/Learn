FreeRTOS Queues: Real API

A compact reference for task-to-task IMU sample queues.

Create

QueueHandle_t queue =
    xQueueCreate((UBaseType_t)3, sizeof(ImuSample));

if (queue == NULL)
{
    /* Queue was not created. Do not use the handle. */
}

Send From A Task

ImuSample sample = read_imu_sample();

BaseType_t result =
    xQueueSend(queue, &sample, (TickType_t)0);

if (result == pdPASS)
{
    sent_sample_count++;
}
else
{
    dropped_sample_count++;
}

Receive In The Owner Task

ImuSample received;

if (xQueueReceive(queue, &received, portMAX_DELAY) == pdPASS)
{
    process_imu_sample(&received);
}

Return Values

CallSuccessFailure means
xQueueCreateNon-NULL handleQueue not created, often allocation/configuration failure.
xQueueSendpdPASSNo space became available before the block time expired.
xQueueReceivepdPASSNo item became available before the block time expired.

Queues copy item bytes. Use &sample to copy the sample into the queue and &received to copy the next item out. Queue pointers only when pointer ownership and lifetime are deliberately designed.
