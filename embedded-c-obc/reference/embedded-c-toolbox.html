Qualifiers And Control Words

Tool Meaning Firmware posture
constRead-only through this name.Use on input buffers and tables you must not mutate.
volatileEach access is hardware/compiler-visible.Use for memory-mapped registers, not synchronization.
staticFile-private at file scope; static lifetime inside functions.Use for private helpers and module-local state.
_AtomicAtomic object for C11 concurrency.Use only with platform/toolchain understanding; often RTOS primitives are clearer.
restrictNo-alias promise for pointer-accessed objects.Avoid until you can prove the contract.
NULLPointer to no object.Check before dereferencing API pointer inputs.

Review rule: uint8_t is data. size_t is how much data. A pointer is where data starts. An array is storage. volatile is hardware visibility.

Parser Pattern

static bool parse(const uint8_t *frame, size_t frame_len, Packet *out)
{
    if (frame == NULL || out == NULL) {
        return false;
    }

    if (frame_len < MIN_FRAME_LEN) {
        return false;
    }

    /* Now fixed indexes proven by MIN_FRAME_LEN may be read. */
}
