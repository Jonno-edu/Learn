Lesson 0007: Pack IMU Telemetry

A sensor struct is typed C data. A telemetry packet is bytes. This lesson teaches the careful translation between them without overrunning the caller's output buffer.

The One Thing

An array owns storage. A pointer points at storage. A length tells you how much storage is valid. A packet-packing function must use all three facts before writing bytes.

OBC habit: before writing out_frame[i], prove out_capacity is large enough for the whole frame.

The Function Contract

bool pack_imu_packet(const ImuSample *imu,
                     FlightState state,
                     uint8_t *out_frame,
                     size_t out_capacity,
                     size_t *out_len);
Parameter Contract
const ImuSample *imuAddress of read-only typed sensor data.
FlightState stateNamed flight state to encode as one byte.
uint8_t *out_frameAddress of caller-owned byte storage to write.
size_t out_capacityHow many bytes are valid in out_frame.
size_t *out_lenAddress where the function reports how many bytes it wrote.

Packet Format

The exercise packs one IMU sample into a small OBC telemetry frame:

0SOF00xa5
1SOF10x5a
2message ID0x42
3payload length13
4flight stateone enum value
5..16six signed IMU fieldseach int16_t, little-endian
17checksumlow byte sum from message ID through payload

Little-Endian Packing

A packet is bytes, so a signed 16-bit value must be split into two bytes. This exercise uses little-endian order: low byte first, high byte second. 

uint16_t bits = (uint16_t)value;
out_frame[index] = (uint8_t)(bits & 0xffu);
out_frame[index + 1u] = (uint8_t)((bits >> 8u) & 0xffu);
Important: do not write bytes as you discover problems. Reject bad arguments and insufficient capacity first, then pack the frame once the output buffer is known to fit.

Practice

Open exercises/0007-pack-imu-telemetry/. Implement the helper that packs signed 16-bit values and the packet builder.

  1. Reject null pointers.
  2. Reject invalid flight states.
  3. Check out_capacity before writing frame bytes.
  4. Pack all int16_t values low byte first.
  5. Compute checksum over msg_id, payload_len, and payload bytes.
  6. Set *out_len only after a complete frame is written.
